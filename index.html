Лабораторная работа 2

Роль в команде Empty: системное прораммирование (СП)

Команда приняла участие в конкурсе с данным проектом

Репозиторий проекта доступен по ссылке: Stankin Board

Проект и его описание доступны по ссылке: Landing Page

Стек технологий:

Блок навигации содержит:

  • Логотип приложения (ссылка на главную страницу)
  • Вкладка «О проекте» - т.е. Landing Page

На верхней панели находятся быть кнопки:

  • Создать объявление
  • Войти

На главной странице пользователю предоставлен следующий функционал:

  • Просмотр объявлений
  • Авторизация с помощью аккаунта Google
  • Заполнение формы создания объявления

Landing Page включает:

  • Текстовое описание приложения
  • Плитки с информацией об участниках команды

Страница авторизации содержит:

  • Поля для ввода логина и пароля
  • Кнопки входа с помощью Google аккаунта

Используемые средства разработки ПО

  • Firebase – для взаимодействия с БД и авторизации пользователей, а также для деплоя приложения
  • HTML, CSS, JavaScript
  • Фреймворк Bootstrap для адаптирования

Совместная разработка ПО и контроль изменений

  • Распределённая система управления версиями Git;
  • Trello – программа для управления проектами
